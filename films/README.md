# Tintash Movies (React FrontEnd)

## Setup

### Clone Latest API Repository

```
git clone https://github.com/moizwasti/react-api-test
```

### Clone This Repo

```
git clone https://github.com/moizwasti/films-react
```

### Run

#### Run in dev mode

```
npm start
```

#### Run in production mode

```
npm run build
serve -s build
```
